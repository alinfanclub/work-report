# 나만의 작업일지 프로젝트

나만의 작업일지 프로젝트는 회사에서 작업일지를 엑셀로 정리해서 전달하고자 하는데 보안상의 이유로 엑셀 프로그램을 사용할 수 없거나 회사 컴퓨터에 파일이 있다보니 외부에서 긴급하게 수정 및 전송을 하기 위해 만들었다

## 사용한 기술
* React.js
* Tailwinds
* Reacticons
* React-router-dom
* React-query
* Handsontable
* XLSX
* timeago.js
* uuid
